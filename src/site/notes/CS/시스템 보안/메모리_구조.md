# Register 크기별 분류
## 8비트 register
- ah : 왼쪽 8비트 상위(high) 부분
- al : 오른쪽 8비트 하위(low) 부분
## 16비트 register : ax, bx, cx, dx
## 32비트 register : eax, ebx, ecx, edx
- e는 **extended**를 의미한다.
